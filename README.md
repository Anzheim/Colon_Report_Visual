### 1. 找個新地方下載專案
在終端下載
```
git clone https://github.com/Anzheim/Colon_Report_Visual.git
```
或下載 .zip 壓縮檔
### 2. 建立虛擬環境
```
python -m venv [虛擬環境名稱]
```
或這個
e.g. 
python -m venv venv
### 3. 進入虛擬環境
```
[虛擬環境名稱]\Scripts\activate
```
e.g. `venv\Scripts\activate`
### 4. (補充)離開虛擬環境
```
deactivate
```
或
```
exit
```
---
### 套件下載
```
pip install -r requirements.txt
```
