### 1. 找個新地方下載專案
在終端下載
```
git clone https://github.com/Anzheim/Colon_Report_Visual.git
```
或下載 .zip 壓縮檔
### 2. 建立虛擬環境
```
virtualenv [虛擬環境名稱]
```
e.g. `virtualenv venv`

如果失敗，試這個
```
python -m venv [虛擬環境名稱]
```
或這個
e.g. 
py -3 -m venv venv
### 3. 進入虛擬環境
```
[虛擬環境名稱]\Scripts\activate
```
或
```
[虛擬環境名稱]\Scripts\activate.bat
```
e.g. `venv\Scripts\activate`
### 4. (補充)離開虛擬環境
```
deactivate
```
或
```
exit
```
